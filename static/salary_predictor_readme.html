🎯 Project Overview

This salary prediction system uses Random Forest Regression to analyze and predict salaries based on:

Analyzes the impact of educational qualifications on salary predictions

⏰ Years of Experience

Considers professional experience in salary calculations

📍 Location

Factors in urban, suburban, and rural location differences

💼 Job Title

Accounts for different job positions and responsibilities

👤 Age & Gender

Incorporates demographic factors in predictions

🚀 Installation & Setup

Clone the Repository

git clone https://github.com/yourusername/salary-predictor.git cd salary-predictor

Create Virtual Environment

python -m venv venv source venv/bin/activate # On Windows: venv\Scripts\activate

Install Dependencies

pip install flask pandas scikit-learn matplotlib seaborn numpy

Prepare Dataset

Ensure salary_prediction_data.csv is in the root directory with required columns.

Run the Application

python app.py

The application will be available at http://localhost:5000

📊 Dataset Requirements

Column Data Type Description Valid Values
Education String Educational qualification High School, Bachelor, Master, PhD
Experience Integer Years of experience 0-70
Location String Work location type Urban, Suburban, Rural
Job_Title String Job position Director, Manager, Analyst, Engineer
Age Integer Age in years 18-100
Gender String Gender Male, Female
Salary Float Monthly salary (target) Positive numbers
